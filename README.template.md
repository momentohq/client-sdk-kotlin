## Packages

The Kotlin SDK is available on Maven Central:

### Gradle

```kotlin
dependencies {
    implementation("software.momento.kotlin:sdk:0.1.3")
}
```

### Maven

```xml
<dependency>
    <groupId>software.momento.kotlin</groupId>
    <artifactId>sdk</artifactId>
    <version>0.1.3</version>
</dependency>
```
